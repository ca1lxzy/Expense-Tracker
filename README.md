# 💰 Expense Tracker

A simple expense tracking application built with Python to help users manage their daily spending.

## ✨ Features

- Add new expenses
- View all expenses
- Calculate total expenses
- Easy-to-use interface

## 🛠️ Technologies

- Python 3

## 🚀 Installation

```bash
git clone https://github.com/ca1lxzy/Expense-Tracker.git
cd Expense-Tracker
python main.py
```
